## Overview

This repository contains a personal portfolio website designed to highlight:

- A professional introduction and personal summary
- Education details and technical skill highlights
- Featured projects in software engineering, AI, and web development
- Contact information and a simple inquiry form
- Social links to GitHub and LinkedIn

The site is built with plain HTML, CSS, and Bootstrap-based layout components, making it lightweight, easy to customize, and simple to host.

## Features

- Responsive one-page layout
- Navigation menu for quick access to key sections
- Hero section with personal introduction and external profile links
- Education section including university details and GPA
- Skills section covering programming languages and web technologies
- Projects section showcasing multiple portfolio items
- Contact section with email, phone, location, and a message form
- Custom dark visual theme with warm accent colors and modern typography

## Technologies Used

- HTML5 for page structure
- CSS3 for custom styling and layout
- Bootstrap for responsive components and grid system
- Font Awesome for icons
- Google Fonts for typography
- JavaScript and jQuery libraries for Bootstrap interactions

## Project Structure

- index.html - Main structure of the portfolio website
- css/style.css - Custom styling and theme design
- css/bootstrap.css - Bootstrap stylesheet
- js/bootstrap.js - Bootstrap JavaScript functionality
- js/jquery-4.0.0.min.js - jQuery library
- js/popper.min.js - Popper.js dependency for Bootstrap
- imgs/ - Folder for project assets and images
- README.md - Project documentation

## Getting Started

This project is a static website, so no complex build process is required.

### Option 1: Open directly in a browser

- Open index.html in your preferred browser.

### Option 2: Run a local server

If you want a more reliable local preview, run:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```
